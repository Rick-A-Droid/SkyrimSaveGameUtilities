Skyrim Potion Maker Help Page
ON/OFF Button

       The function is OFF.  Click it to start it.

       The function is ON.  Click it to stop it.
Your PC's IP Address

Your PC's IP Address is displayed here.  This is the IP you need to enter on the Network Setup page of the Skyrim Potion Maker app.
Port:

This is a 4-digit number of the Port that the PC will listen on.  It needs to be the same number as the one entered on the Network Setup page of the Skyrim Potion Maker app.
Check Boxes

Here you can set what types of saves the PC looks for.  It will never look for autosaves.

  • Quicksaves - If checked, quicksaves will be used (it does not matter if Savior is also running and renaming them).
  • Fullsaves - If checked, the fullsaves will be used.
  • Other saves - If you are using other mods that save the game with another filename, enter the first part of the filename (that does not change) here, and check the box.
Status Window

The window displays messages to indicate its progress.
WIFI Connection

If you are connecting your Android phone to your system's WIFI, you may need to set your broadband router to send unsolicited traffic to your PC.  On my Netgear broadband router, that I use with AT&T broadband, I found the setting under Advanced -> Connection Configuration.

Where it says: Current settings for unsolicited traffic is Dropped. (which is the default), I changed it to IP Passthrough in the drop-down box.

In Please select the device to receive all unsolicited traffic I chose the name of my PC.

In IP Address: I entered the IP address displayed in Your PC's IP Address

Although this worked for me, I do not know if this opens any possible security issues in allowing unsolicited traffic into the PC.  You should read the documentation that came with your router to determine what to do and consider if you can add any additional security options to it.

Personally, I disable the unsolicited traffic thing in my router when I am not playing the game.
USB Connection

This is the easiest way to connect your phone to your PC and does not require any additional work to be done to get the communcations between the phone and PC to work. 

How it works

When the Directory Watcher detects that a new file has been created in the Savegame directory, it wakes up this program.  If the filename is one of the ones selected (quicksave, fullsave, and/or another type) the program reads the savegame file to extract the alchemy information.  It builds a ZIP file in the /spm_output directory, named spm_out.zip

The ZIP file contains two files.  The first is a text file with a response string.  All responses from the PC have this file.  The text is used by the Android app to determine what the response from the PC is.

The second file is an XML file containing the following:

  • The name of your character.
  • The DLCs and Mods found that have added Ingredients and/or effects to the game.
  • The Ingredients (by their BaseID) and a flag that indicates whether you have discovered each of the Ingredient's 4 effects.

This data is used by the Android app to look up the actual Ingredient (using its BaseID) from the ingredients information that the app has in its XML files.  If an Ingredient cannot be found, it will display an error message.  If this happens you need to edit the app's XML files to add any additional ingredients added or changed by other mods.  See the help files in the Skyrim Potion Maker app.

If you are doing alchemy in the game, discovering other effects, and wish to update the Android app, you will need to do another savegame, and then retrieve the updated data from the app.
