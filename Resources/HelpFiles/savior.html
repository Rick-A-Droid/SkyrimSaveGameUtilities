Savegame Savior Help Page
ON/OFF Toggle Button
Savegame Savior is OFF.  Click it to start it.
Savegame Savior is ON.  Click it to stop it.

Number of saves:

Enter the number of saves you want to keep.  The minimum is 3.
How it works

The program uses a Directory Watcher to wait until a new file is created in the savegame directory.  It looks specifically for the quicksave.ess file to be created.

However, this filename is not hard-coded.  I put the filenames and extensions used by this program in the Language file, just in case they might be different in other languages.

Each time the program detects a new quicksave, it will rename the quicksave.ess and quicksave.skse (if you installed SKSE) files to a filename generated from two things:

The first part comes from the following item in the language file:  <string key="savior_filename_prefix" value="gamesave_" />

You can modify the gamesave_ part to anything you would like (that includes characters allowed in a filename).

The scond part is the system date and time the save occurred.  Here are examples of the filenames created:

  • gamesave_Fri Jan 24 15_39_34 PST 2014.ess
  • gamesave_Fri Jan 24 15_39_34 PST 2014.skse

The program keeps track of all of the renamed files, and when it reaches your Number of saves limit, it deletes the oldest one when renaming a new one.

One Problem:

I found that the game will sometimes fail to automatically reload one of these saves when I die.  It will just stay on the screen showing my dead body.  So, I just have to do the reload manually, by hitting the ESC key and loading the last save.
